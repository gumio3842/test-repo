
pwd
	현재 디렉토리 표시

cd
	현재 위치를 이동. cd 단독 사용 시 홈으로 이동하고, 뒤에 경로를 입력 시 해당 경로로 이동.
	경로 입력 시 /로 시작하면 절대경로, 파일명으로 시작하면 상대경로인데, 상대경로의 경우 '..'이  상위 디렉토리를 의미함.
	사용법: cd  [경로]

ls 
	디렉토리 안에 있는 컨텐츠들( 파일 및 하위 디렉토리) 표시.
	옵션이 다양하게 존재하는데, 명령어 뒤에 옵션을 추가해서 사용하면 옵션에 따른 결과가 출력됨.
	사용법: ls [옵션] [경로]
	>>옵션
		-a: 숨겨진 파일들을 전부 표시. 
		-d: 디렉토리 자신에 대한 정보를 표시. 안에 들어있는 컨텐츠들에 대한 정보를 표시하는것이 아님.
		-i: inode[1] 번호를 파일 명 앞에 표시.
		-l: 파일에 대한 자세한 정보[2]를 표시. 
		-A:  -a에서 '.'과 '..'을 제외한 파일들을 표시. '.'은 현재 디렉토리, '..'은 상위 디렉토리를 의미.
		-F: 파일 타입을 기호로 표시. '/'는 디렉토리, '*'는 실행파일, '@'는 심볼릭 링크.[4]
		-L: 뒤 경로가 가리키는 파일이 심볼릭 링크여도, 심볼릭 링크가 가리키는 경로를 표시하는것이 아닌 경로의 파일의 컨텐츠를 표시.
		-R: 하위 폴더를 전부 출력. 경로가 가리키는 디렉토리 기준으로 하위의 모든 디렉토리의 내용물과 그 산하의 디렉토리의 정보들까지 전부 출력하는것.

		[1] 파일에 대한 정보를 담고있는 구조체. 번호는 구조체의 이름 같은것. 파일명은 inode의 라벨같은것.
		[2] 파일 권한 | 하드링크[3] 수 | 소유자 | 그룹 | 파일 크기 | 수정 날짜 | 파일 이름 순서
		[3] inode에 파일명을 붙이는것. 파일명 갯수는 하드링크 수.
		[4] 바로가기. 하드링크와는 다르게 파일 경로를 저장하고, 그 위치로 이동하는 방식. 
		
MKDIR
	디렉토리를 생성. 기본적으로 현재 디렉토리 바로 하위의 디렉토리만 생성 가능.
	사용법: mkdir [옵션] [경로]
	>>옵션
		-p: 생성하려는 디렉토리의 상위 디렉토리까지 자동 생성.  즉, 디렉토리 아래의 디렉토리 아래의 디렉토리 같은것을 만드는것이 가능해짐.

rmdir
	비어있는 디렉토리를 삭제. 내부에 파일이 있다면 삭제되지 않음. 디렉터리 외에는 삭제 불가.
	사용법: rmdir [옵션] [경로]
	>>옵션
		-p: 삭제하려는 디렉토리의 상위 디렉터리도 동시에 삭제. 맨 안쪽에 있는 디렉토리부터 순차적으로 삭제되며, 만약 순차적으로 삭제되는 과정에 비어있지 않은 디렉토리가 존재한다면 삭제되지 않음.
		
cat
	화면에 컨텐츠의 내용을 출력.
	사용법: cat [옵션] [파일]
	>>옵션
		-n : 줄 번호 출력.  줄이 바뀔때마다 줄 맨 앞에 숫자 출력됨.

more
	화면의 컨텐츠의 내용을 한 페이지씩 출력.
	사용법: more [옵션] [파일명]
	>>조작법
		스페이스바: 한 페이지 추가 출력
		엔터: 한 줄 추가 출력
	>>옵션
		+[줄 번호]: 입력된 줄 번호의 줄부터 시작.
		
less
	화면에 컨텐츠의 내용을 한 화면분량으로 출력. 사실상 more 상위호환 느낌.
	사용법: less [파일명]
	>>조작법
		j, ↓: 한 줄 아래로 이동.
		k, ↑: 한 줄 위로 이동.
		스페이스바, ^f: 다음 페이지 이동.
		^b: 이전 페이지 이동.
		/[텍스트]: 텍스트의 내용 검색.
		n: 다음 검색결과로 이동.
		N: 이전 검색결과로 이동.
		q: 종료.

tail
	파일의 마지막 n줄을 출력.  기본값은 10줄.
	사용법: tail [옵션] [파일명]
	>>옵션
		-n [숫자]: 출력 할 줄 수 조정. -n +[숫자], -[숫자] 형태도  동일한 효과.
		+[줄 번호]: 입력된 줄 번호의 줄 부터 끝까지 출력.
		-f: 파일 변화 실시간 모니터링. 파일이 갱신되면 새 줄이 자동으로 출력.
		
cp
	파일을 복사. 복사 될 위치에 파일이 이미 있다면 자동으로 덮어쓰여짐.
	사용법: cp [옵션] [복사 할 파일명 또는 디렉토리] [복사 될 파일명 또는 디렉토리]
	>>옵션
		-i: 복사 될 위치에 파일이나 디렉토리가 이미 존재하는 경우, 덮어쓰기 여부를 물어봄.
		-r: cp가 디렉토리도 복사 할 수 있게 함. 이 옵션을 사용하지 않으면 디렉토리는 복사 불가능하지만, 이 옵션이 있다고 파일을 복사 할 수 없는것은 아님.

mv
	파일을 이동하거나 이름을 변경. 이동 할 위치에 파일이 이미 있다면 자동으로 덮어쓰여짐. cp와는 다르게 옵션을 넣지 않아도 디렉토리도 이동시킬 수 있음.
	사용법: mv [옵션] [이동 할 파일의 경로, 또는 이름을 변경 할 파일명 또는 디렉토리] [파일이 이동 될 경로, 또는 이름이 변경 될 파일명 또는 디렉토리명]
	>>특이사항
		파일을 이동하는 경우, [이동 할 파일의 경로, 또는 이름을 변경 할 파일명 또는 디렉토리] 파트는 여러개여도 상관없음. 
	>>옵션
		-i: 이동 할 위치에 파일이나 디렉토리가 이미 존재하는 경우, 덮어쓰기 여부를 물어봄.
		
rm
	파일을 삭제.
	사용법: rm [옵션] [삭제할 파일 또는 디렉토리명]
	>>옵션
		-i: 파일을 정말 삭제할지 여부를 물어봄.
		-r: 디렉토리를 삭제할때 사용됨. 이 옵션을 사용하지 않으면 디렉토리는 삭제 불가능하지만, 이 옵션이 있다고 파일을 삭제 할 수 없는것은 아님.
		
ln
	파일에 링크를 추가. 기본값은 하드링크.
	사용법: ln [옵션] [파일명] [링크 이름]
	>>옵션
		-s: 파일에 링크를 거는 방식을 심볼릭 링크로 변경함.함
		
touch
	빈 파일을 생성. 파일이 이미 존재한다면 타임스탬프(접근 시간, 수정 시간)만 변경함.
	사용법: touch [옵션] [생성 할 파일명 또는 타임스탬프를 수정할 파일명]
	>>특이사항
		-a와 -m을 같이 사용하면 옵션이 없는 touch와 동일한 동작을 함.
		-r와 -t는 동시에 사용 불가
	>>옵션
		-a: 파일의 타임스탬프를 수정할 때 파일의 접근 시간만 변경함. 
		-c: touch로 빈 파일을 생성하지 못하게 함.
		-m: 파일의 타임스탬프를 수정할 때 파일의 수정 시간만 변경함.
		-t [타임스탬프[1]]: 생성하거나 변경할 파일의 타임스탬프를 특정 시각으로 지정.
		-r [복사 할 파일명]: 생성하거나 변경할 파일의 타임스탬프를 다른 파일에서 복사.
		
		[1] 타임스탬프 지정 방법은 다음과 같음. [[CC]YY]MMDDhhmm[.ss]. [[세기]연도]  월   일   시   분  [. 초]라는 뜻이다. ex.) 202512100358.25 => 2025년 12월 10일 03시 58분 25초.
		
grep
	파일의 텍스트에서 특정 패턴이 있는 줄을 검색. 패턴은 단순 검색어라 생각해도 됨.
	사용법: grep [옵션] [패턴] [파일명]
	>>옵션
		-i: 대소문자 구분 없이 검색함.
		-l: 해당 패턴을 포함하는 파일의 이름만 출력. 보통 파일명에 *[.확장자]를 넣어 특정 확장자 파일을 검색하는 경우에 사용됨.
		-n: 검색 된 줄의 번호를 출력.
		
find
	특정 위치에서 검색 조건에 맞는 파일을 검색.
	사용법: find [경로] [검색조건]  [동작]
	>>검색조건
		-name "패턴": 파일 이름이 패턴과 일치하는 파일 검색. 일치해야함.
		-type [타입 코드]: 파일 종류가 타입 코드와 일치하는 파일 검색. 파일 코드는 다음과 같음.  f: 일반 파일, d: 디렉토리, l: 심볼릭 링크
		

Linux 텍스트 에디터 분류
	Line Editor (ed, ex, sed) / Screen Editor (vi, emacs) / GUI Editor (gedit)
	vi는 모달(Mode 기반) 에디터라서, 입력(Insert/Input)과 명령(Command)을 모드로 구분해서 쓴다.

vi
	터미널 기반 화면 에디터 vi 실행. 파일을 열어 편집함. 파일명을 비우면 나중에 파일명을 정할 수 있음.
	사용법: vi [파일명]
	>>모드키
		i: 현재 커서 위치에 입력을 시작함.
		a: 현재 커서 다음 칸 부터 입력을 시작함.
		o: 현재 줄 아래에 새 줄을 만들고 입력함.
		I: 현재 커서 줄 맨 앞(첫 칸)으로 가서 입력함.
		A: 현재 줄 맨 끝으로 가서 입력을 시작함.
		O: 현재 줄 위에 새 줄을 만들고 입력함.
		Esc: 입력모드에서 명령모드로 복귀.
		
		:q : 변경 사항이 없을 때 종료 함.
		:q! : 변경 사항이 있을 때 강제 종료 함.
		:w [파일명]: 변경 사항을 저장함. 파일명을 입력하면 다른 이름으로 저장.
		:wq : 저장하고 종료함. !를 붙여도 동작.
		ZZ: 저장하고 종료함.
		
		h: 왼쪽으로 이동함.
		j: 아래쪽으로 이동함.
		k: 위쪽으로 이동함.
		l: 오른쪽으로 이동함.
		^: 현재 줄 시작점으로 이동함.
		$: 현재 줄 끝점으로 이동함.
		-: 이전 줄 시작점으로 이동함.
		+ or Enter: 다음 줄 시작점으로 이동 함.
		H: 화면의 맨 위로 이동함.
		M: 화면의 중간으로 이동함.
		L: 화면의 맨 아래로 이동함.
		w: 다음 단어의 시적점으로 이동함.
		e: 다음 단어의 끝점으로 이동함.
		b: 이전 단어의 시작점으로 이동함.
		
		Ctrl+u: 화면을 반 화면 만큼 위로 스크롤 함.
		Ctrl+d: 화면을 반 화면 만큼 아래로 스크롤 함.
		Ctrl+b: 화면을 한 화면 만큼 위로 스크롤 함.
		Ctrl+f: 화면을 한 화면 만큼 아래로 스크롤 함.
		Ctrl+y: 화면을 한 줄 만큼 위로 스크롤 함.
		Ctrl+e: 화면을 한 줄 만큼 아래로 스크롤 함.

		G: 파일의 마지막 줄로 이동함.
		[줄 번호]G: 줄 번호에 해당하는 줄로 이동함.
		:$ : 파일의 마지막 줄로 이동함.
		:[줄번호] : 줄 번호에 해당하는 줄로 이동함.
		
		r[교체 할 문자]: 커서가 위치한 곳의 문자를 다른 문자로 교체함.
		[교체 할 단어 갯수]cw: 커서가 위치한 곳의 단어를 삭제하고 입력 모드로 변경함. 갯수를 입력했다면 그만큼 단어를 삭제.
		[교체 할 문자 갯수]s: 커서가 위치한 곳의 문자를 삭제하고 입력 모드로 변경함. 갯수를 입력했다면 그만큼 문자를 삭제.
		cc: 현재 줄 전체를 삭제하고 입력 모드로 변경함.
		C: 커서부터 줄 끝까지 삭제하고 입력 모드로 변경함.
